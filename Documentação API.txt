- API de comando de voz

Quanto ao comando de voz para utilizar no aplicativo, a primeira ideia era que fossem pré-estabelecidos comandos de voz que, quando ouvidos, já executavam determinadas ações. 

A API mais próxima encontrada para isso foi o Google Voice Actions, porém esta API escuta apenas ações já pré-determinadas. Segue a lista:
https://developers.google.com/voice-actions/system/#system_actions_reference.

Minha sugestão é que utilizemos a api de Speech to text da própria Google como ouvinte para o input de voz, armazenando o texto resultante da voz em string e comparando com os métodos existentes. O único modelo a ser utilizado será o command_and_search.
https://cloud.google.com/speech-to-text/
